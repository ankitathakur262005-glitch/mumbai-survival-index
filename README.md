# Mumbai Survival Index

## Project Overview
This project analyzes housing prices, fresher salaries, and living costs to evaluate whether students and freshers can afford living in Mumbai.

## Problem Statement
Mumbai is one of the most expensive cities in India. Many students and freshers struggle with high living expenses and housing costs.

## Datasets Used
- Mumbai Housing Dataset
- Indian Job Market Dataset
- Living Cost Dataset

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Mumbai has extremely high living costs
- Most fresher salaries fall in low-income categories
- Median fresher salary is lower than average living cost
- Housing inequality is significant across locations

## Survival Index
The project compares median fresher salary against average living cost to evaluate affordability.

## Conclusion
The analysis suggests that independent living in Mumbai can be financially difficult for many students and freshers.
